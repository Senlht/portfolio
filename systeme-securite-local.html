Système de Sécurité Local

Contrôle d'accès via clavier matriciel

Description du Projet

Conception et implémentation d'un système de sécurité robuste pour le contrôle d'accès à une chambre sécurisée. Le système utilise un clavier matriciel pour la saisie du code et intègre des mécanismes de sécurité avancés.

Fonctionnalités Principales

  • Saisie de code sécurisé via clavier 4x4
  • Verrouillage automatique après 3 tentatives échouées
  • Indication LED pour le statut du système
  • Alarme sonore en cas d'intrusion
  • Possibilité de changer le code d'accès

Architecture du Système

Le système repose sur une architecture modulaire avec un microcontrôleur central gérant l'interface utilisateur, la logique de sécurité et le contrôle du mécanisme de verrouillage.

Innovations

Implémentation d'un algorithme de détection de tentative de forcing et d'un système de temporisation intelligent pour prévenir les attaques par brute force.

Technologies Utilisées

Arduino Uno Clavier 4x4 Servomoteur Buzzer LED RGB C++ Électronique

Durée

6 semaines
