Robot Autonome

Navigation intelligente et évitement d'obstacles

Description du Projet

Conception et réalisation d'un robot mobile autonome capable de naviguer dans un environnement inconnu tout en évitant les obstacles. Développé dans le cadre du club de robotique ROBOTEK.

Fonctionnalités Principales

  • Détection d'obstacles via capteurs ultrasoniques
  • Algorithme de navigation autonome
  • Suivi de ligne avec capteurs infrarouges
  • Communication sans fil pour le monitoring
  • Batterie LiPo avec gestion de l'autonomie

Algorithmes Implémentés

Développement d'un algorithme de prise de décision en temps réel combinant les données des capteurs pour une navigation fluide et sécurisée. Implémentation de comportements réactifs et proactifs.

Performances

Le robot démontre une capacité à naviguer dans des environnements complexes avec un taux de réussite de 95% dans l'évitement d'obstacles et une précision de suivi de ligne de 98%.

Technologies Utilisées

Arduino Mega Capteurs HC-SR04 Moteurs DC L298N Capteurs IR Bluetooth C++ ROS

Durée

3mois
